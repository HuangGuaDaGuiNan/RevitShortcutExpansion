# RevitShortcutExpansion
An extension tool for Revit shortcut keys

- 以前在办公室总要去同事电脑排查问题，或者有时要用公用电脑进行Revit演示，但不同使用者的快捷键方案不一样

- 而Revit的快捷键管理只能通过手动导入导出的方式进行快捷键的批量替换

- 通过这个扩展工具，Revit可以保存多套快捷键方案，并可以很方便地进行切换
